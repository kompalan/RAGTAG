# RAGTAG

RAGTAG is an experiment to test whether explicit structure helps LLM reliability relative to just text in correctness-sensitive areas, like coding.

For the initial use-case of code generation in large repos, the goal is to see whether giving the LLM a representation of the repository as a dependency graph (shown below), helps:
